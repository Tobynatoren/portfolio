# Design Context

## Target Audience
- Recruiters/hiring managers (skim in 30 seconds)
- Technical leads/interviewers (dig into code and design decisions)

## Brand Personality
Curious builder — works across wildly different domains (game dev, sports data, web apps, trucking logistics). Playful but competent. The site should feel handmade and personal, not generated or templated.

## Aesthetic Direction
Inspired by joshwcomeau.com — warm dark theme with colorful playful accents, text flowing freely on the page (not boxed in cards), whimsical micro-interactions. Content breathes. Nothing feels "trapped."

## Key Principles
- **No boxes** — content lives on the page directly, differentiated by typography and spacing, not containers
- **Multi-color accents** — different colors for different projects/sections, reflecting the range of work
- **Organic layout** — asymmetric, varied spacing, break the grid intentionally
- **Playful motion** — spring physics, interactive hovers, staggered reveals
- **Text-first** — typography carries the design, not decorative containers
- **Handmade feel** — the site should look like someone built it with care, not like a template

## Color System
- Background: near-black with slight warm tint (NOT navy)
- Surface: subtle elevation for code blocks and necessary containers only
- Text: warm near-white (not pure #fff)
- Multiple accent colors: warm gold (primary), coral, teal, lavender — each project/section can have its own
- Borders: minimal, warm gray, only where structurally needed

## Typography
- Display/headings: Space Grotesk (geometric, builder energy)
- Body: DM Sans (clean, warm, not Inter)

## Anti-Patterns to Avoid
- Card grids with identical cards
- Indigo/purple on dark navy (AI palette)
- Glassmorphism / backdrop blur
- Radial glow blobs
- Same spacing everywhere
- Redundant section subtitles
- Everything centered
- Inter font
